1.通过github将音乐伴侣下载到扩展区

2.下载对应的插件st api wrapper

3.打开你部署的酒馆配置config.yml

路径一般在./SillyTavern/config.yml

然后查找

enableServerPlugins

项,将它从false改成true(这是配置st-wrapper后端插件的程序之一),改完yml以后一定要重启你的酒馆

然后跑到你的扩展文件夹,一般是酒馆根目录/data/default-user/extensions,找到这个
